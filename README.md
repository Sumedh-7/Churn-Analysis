# Customer Churn Prediction & Risk Segmentation Dashboard

## Project Overview
This project predicts telecom customer churn using machine learning models and segments customers into different churn-risk categories.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- Matplotlib
- Seaborn
- Plotly

## Features
- Exploratory Data Analysis
- Feature Engineering
- Multiple ML Model Comparison
- ROC Curve Evaluation
- Risk Segmentation
- Interactive Visualizations

## Models Used
- Logistic Regression
- Random Forest
- Gradient Boosting

## Project Structure
- analysis.ipynb
- charts/
- requirements.txt
- summary.docx
- model_comparison.png

## Outcome
The Random Forest model achieved the best performance and helped identify high-risk customers for proactive retention strategies.
